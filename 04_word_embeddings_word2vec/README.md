# Word Embeddings - Word2Vec

This module covers Word2Vec algorithms for learning word embeddings.

## Contents

- `cbow_from_scratch.ipynb` - Continuous Bag of Words implementation
- `skipgram_from_scratch.ipynb` - Skip-gram implementation
- `negative_sampling.ipynb` - Negative sampling technique
- `embedding_visualization.ipynb` - Visualizing word embeddings

## Learning Objectives

- Understand word embeddings and their importance
- Implement CBOW and Skip-gram models
- Learn negative sampling optimization
- Visualize and analyze word embeddings
