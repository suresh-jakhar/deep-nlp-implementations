# Duplicate Question Detection

This module covers techniques for detecting duplicate questions and measuring text similarity.

## Contents

- `text_similarity_metrics.ipynb` - Various text similarity metrics
- `quora_duplicate_questions.ipynb` - Duplicate question detection on Quora dataset
- `failure_cases.ipynb` - Analyzing failure cases

## Learning Objectives

- Understand text similarity metrics
- Build duplicate detection systems
- Analyze model failures
- Improve model performance
