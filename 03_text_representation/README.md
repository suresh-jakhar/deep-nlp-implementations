# Text Representation

This module covers techniques for representing text as numerical vectors.

## Contents

- `bag_of_words_from_scratch.ipynb` - Implementing Bag of Words from scratch
- `tfidf_from_scratch.ipynb` - Implementing TF-IDF from scratch
- `sklearn_vectorizers_comparison.ipynb` - Comparing with scikit-learn implementations

## Learning Objectives

- Understand Bag of Words representation
- Implement TF-IDF algorithm
- Compare custom implementations with libraries
