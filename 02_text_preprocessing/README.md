# Text Preprocessing

This module covers various text preprocessing techniques essential for NLP tasks.

## Contents

- `text_cleaning.ipynb` - Cleaning and preparing raw text
- `tokenization_from_scratch.ipynb` - Building tokenizers from scratch
- `stopwords_stemming_lemmatization.ipynb` - Text normalization techniques
- `normalization_experiments.ipynb` - Experimenting with different normalization approaches

## Learning Objectives

- Master text cleaning techniques
- Implement tokenization algorithms
- Understand stemming and lemmatization
- Apply text normalization
