# Deep NLP Implementations

A comprehensive collection of NLP concepts and algorithms implemented from scratch.

## Project Structure

- **00_introduction_to_nlp/** - Introduction to Natural Language Processing
- **01_end_to_end_nlp_pipeline/** - Complete NLP pipeline overview
- **02_text_preprocessing/** - Text cleaning, tokenization, and normalization
- **03_text_representation/** - Bag of Words, TF-IDF implementations
- **04_word_embeddings_word2vec/** - Word2Vec (CBOW, Skip-gram) from scratch
- **05_text_classification/** - Classification algorithms for text
- **06_pos_tagging_hmm/** - Part-of-Speech tagging with Hidden Markov Models
- **07_duplicate_question_detection/** - Text similarity and duplicate detection
- **datasets/** - Sample datasets for experiments
- **utils/** - Utility functions and helpers

## Getting Started

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Navigate to any module and explore the Jupyter notebooks
