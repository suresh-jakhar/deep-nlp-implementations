# Text Classification

This module covers various text classification algorithms and techniques.

## Contents

- `naive_bayes_from_scratch.ipynb` - Naive Bayes classifier implementation
- `logistic_regression_text.ipynb` - Logistic regression for text
- `averaging_word_embeddings.ipynb` - Using averaged word embeddings for classification
- `evaluation_metrics.ipynb` - Classification evaluation metrics

## Learning Objectives

- Implement Naive Bayes classifier
- Apply logistic regression to text data
- Use word embeddings for classification
- Evaluate classification models
