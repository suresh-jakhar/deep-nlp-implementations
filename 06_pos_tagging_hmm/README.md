# POS Tagging with Hidden Markov Models

This module covers Part-of-Speech tagging using Hidden Markov Models.

## Contents

- `hmm_pos_tagger.ipynb` - HMM-based POS tagger implementation
- `viterbi_algorithm.ipynb` - Viterbi algorithm for decoding
- `error_analysis.ipynb` - Analyzing POS tagging errors

## Learning Objectives

- Understand Hidden Markov Models
- Implement HMM for POS tagging
- Learn the Viterbi algorithm
- Perform error analysis
